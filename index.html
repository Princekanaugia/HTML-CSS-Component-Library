@Image

We use "img" tag with attribute ~ src="image-source" / ~ to add image on the webpage. Image source can be image link address on the internet or cloud address or on the same repository. this tag is a self-closing tag. for image in the same Repository under Image folder in the Repo, then image source = /Image/image-name.extension. Prefered image extension is " .svg " file nd not " .png " or " .jpg " file.
